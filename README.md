# Making Connections - Profile Page

A responsive and interactive Profile Page built using **HTML5**, **CSS3**, and **JavaScript (DOM Manipulation)**. This project demonstrates foundational front-end development concepts, user interface layout design, and dynamic page interactions.

## Features

* **Dynamic Profile Name Change:** Clicking the "edit profile" link dynamically updates the user's name on the profile card.
* **Interactive Connection Requests:** Users can accept or reject pending connection requests.
* **Live Counter Updates:** * Clicking either accept ($\checkmark$) or reject ($\times$) decreases the pending "Connection Requests" count.
    * Accepting a request automatically increases the "Your Connections" total counter.
* **Element Removal:** Successfully managed requests are dynamically removed from the active DOM list.
* **Responsive Layout:** A clean, multi-column layout tailored for standard desktop views.

## Technologies Used

* **HTML5:** Structured semantic markup.
* **CSS3:** Custom layouts, sticky navigation, card designs, and absolute positioning.
* **JavaScript (ES6+):** DOM selection, event listeners, and dynamic element manipulation (`closest()`, `remove()`).
* **Font Awesome:** Scalable vector icons for user actions and profile details.
